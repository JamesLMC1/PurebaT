Crea Tu Tarea

    ¿Cómo funciona?

    Esta es una aplicación web sencilla diseñada para ayudarte a organizar tus tareas. Solo tienes que escribir una nueva nota, agregarla, y cuando la completes, haz clic sobre ella para marcarla. Si ya no la necesitas, también puedes eliminarla fácilmente.
